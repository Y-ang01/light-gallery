# 光影收藏夹- 专业图片集管理展示平台
 
## 项目概述 
 
这是一个专业的图片集管理和展示平台，支持RAW格式存储和博客功能?
 
## 技术栈 
 
- **前端**: Vue 3 + TypeScript + Element Plus 
- **后端**: FastAPI + Python 3.14.2
- **数据库*: PostgreSQL 16
- **缓存**: Redis 7
- **部署**: Docker + Nginx 


## 快速构建
- **后端**
- python 3.14.2
- postgresql 16
- redis 7
- docker-compose up -d
- cd backend
- pip install -r requirements.txt
- **前端**
- npm install
- npm run build
- npm run dev
- 注意修改start-dev中环境名称conda activate private-website

## 快速启动
- start-dev.bat  启动开发环境
